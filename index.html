Projects

Medical Data Engineering System

This 9-week group project focused on designing a data engineering pipeline for medical data. The goal was to extract, store, process, and load medical datasets into a single structured system that could be used by healthcare professionals.

We built a centralized database and a web interface that allows doctors to easily search patient records without needing to write SQL queries. Users can search by name or date of birth and access premade queries for things like current medications or treatments.

A big design decision was making the system simple for non-technical users. Doctors are not going to write database queries, so we built a dashboard with predefined searches and safeguards to prevent accidental database changes.

SQL Database Design Backend Web Interface Data Engineering

Software Engineering: Patient Exercise System

This project focused on creating a prototype system where patients could complete exercises while doctors monitor their progress. The system was designed to be fun and motivating, especially for younger users.

The application was written in Java and included a command-line interface with ASCII art to make it more engaging. Patients could interact with virtual pets, gain levels, and unlock accessories while completing exercises assigned by doctors.

I worked on coding and developing the admin panel while also helping with the database structure and account management system.

Java Database CLI Software Engineering

Evolutionary AI Optimization

This individual assignment explored evolutionary algorithms to optimize mathematical problems. The goal was to apply biological concepts like mutation, selection, and survival of the fittest to improve solutions over many generations.

The system was used for tasks such as minimizing functions, solving a job sequencing problem, and generating mathematical formulas that best fit given data points.

The project produced graphs showing the improvement of solutions across generations and demonstrated how parameter tuning and selection strategies significantly influence performance.

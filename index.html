02 Selected Projects
01
Medical Records Database Data Engineering
Group · 9 weeks
+
Overview

Designed and built a unified database system for medical professionals to access patient records. Merged multiple datasets into a single source of truth and built a web dashboard on top of it.

Users & Purpose

Targeting doctors and GPs — non-technical users. Built-in pre-made queries (current medications, active treatments) and search by name or date of birth so no raw SQL was ever needed.

My Contributions

Merged the datasets, designed the query layer, built the interactive website and dashboard, and implemented login/session security with full user interaction logging.

Key Design Decision

Consolidated all data into one database for fast, flexible access. Added input sanitisation guards to prevent accidental destructive SQL commands — because a doctor really shouldn't be able to wipe the database.

Tech Stack
SQL SQLite Python Web Dashboard User Auth Data Merging Logging

03
Evolutionary AI Optimizer Evolutionary AI
Solo · Assignment
+
Overview

Implemented and optimised AI systems using biological evolution principles — applying them to function minimisation, a job sequencing problem, and symbolic regression from data points.

Approach

Built a custom survival-of-the-fittest selection function from scratch. Tuned parameters and selection mechanisms carefully — learning that the right algorithm configuration matters as much as the code itself.

Results

Achieved strong optimisation outcomes, visualised via before/after convergence graphs showing the training trajectory. High performance especially on the function optimisation task.

Key Insight

Correct parameter selection and evolutionary mechanisms are just as impactful as implementation quality — a lesson that applies well beyond AI.

Tech Stack
Python Evolutionary AI Function Optimisation Job Scheduling Symbolic Regression Data Visualisation
